# Superstore-Data-Analysis
Retail sales data analysis using Excel and Power BI, focusing on data cleaning, transformation, and dashboard insights for business decision-making.

## 🧾 Project Overview
This project analyzes a global retail superstore dataset spanning 4 years to uncover key business insights and sales patterns. The dataset was sourced from Kaggle and processed using Excel for data cleaning, analysis, and visualization.

---

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA) on retail sales data  
- Identify trends and patterns in sales over time  
- Analyze city-wise, category-wise, and segment-wise performance  
- Understand customer purchasing behavior  
- Prepare the dataset for potential sales forecasting  

---

## 📂 Dataset Information
- **Source:** Kaggle  
- **Type:** Time-series retail dataset  
- **Duration:** 4 years of sales data  

### 📌 Features include:
- Order Date, Ship Date  
- Customer details (Segment, Region, City)  
- Product details (Category, Sub-Category, Product Name)  
- Sales values  

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables & Charts  
- Power BI 
---

## 🔧 Data Cleaning Process
- Verified and corrected data types (date, numeric, text)  
- Removed duplicates and handled missing values  
- Standardized categorical values for consistency  
- Checked for outliers and ensured data accuracy  

---

## 📊 Analysis Performed
- City-wise sales performance  
- Category and sub-category demand analysis  
- Segment-wise contribution  
- Region-wise comparison  
- Time-based sales trend analysis  

---

## 🔍 Key Insights
- Certain cities contribute significantly higher sales than others  
- A few product categories dominate overall sales  
- Consumer segment drives the majority of sales  
- Regional variations indicate potential growth opportunities  

---

## 🔮 Future Scope
- Time series forecasting to predict next 7 days sales  
- Advanced analysis using tools like Python
- Business strategy recommendations based on insights
